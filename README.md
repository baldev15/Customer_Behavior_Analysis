# 📊 Data Analytics Project

## Overview

This project demonstrates a complete data analytics workflow, starting from data loading and exploration to database analysis and interactive dashboard creation. The goal is to transform raw data into meaningful insights that support data-driven decision-making.

## Dataset

* Dataset loaded and analyzed using Python.
* Includes data cleaning, preprocessing, and validation.
* Prepared for SQL analysis and Power BI visualization.

## Tools & Technologies

* **Python** 
* **Jupyter Notebook**
* **SQL** (PostgreSQL / MySQL / SQL Server)
* **Power BI**
* **Gamma** (Presentation)
* **Git & GitHub**

## Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data types and structure.

### 2. Exploratory Data Analysis (EDA)

* Explored dataset characteristics.
* Generated summary statistics.
* Identified trends, patterns, and relationships.
* Visualized data using charts and graphs.

### 3. Data Cleaning

* Removed duplicate records.
* Handled missing values.
* Corrected inconsistent data.
* Prepared data for analysis.

### 4. SQL Analysis

* Imported cleaned data into PostgreSQL, MySQL, and SQL Server.
* Executed SQL queries to:

  * Filter records
  * Aggregate data
  * Group and sort results
  * Join tables (where applicable)
  * Generate business insights

### 5. Power BI Dashboard

* Connected Power BI to the database.
* Built interactive dashboards.
* Created KPIs and visual reports.
* Added filters and slicers for better analysis.

### 6. Report & Presentation

* Prepared a detailed analytics report.
* Created a professional presentation using Gamma.
* Summarized key findings and recommendations.

## Dashboard Features

* Interactive KPI cards
* Bar, Line, Pie, and Column charts
* Filters and slicers
* Trend analysis
* Business performance insights

## How to Run

1. Clone this repository.
2. Install the required Python libraries (`pandas`, `numpy`, `matplotlib`, `seaborn`).
3. Open the Jupyter Notebook files and run them in order.
4. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
5. Execute the SQL scripts.
6. Open the Power BI `.pbix` file and refresh the data connection.
7. Review the report and presentation for project insights.
